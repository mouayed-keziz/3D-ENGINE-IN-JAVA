# 3D Engine In Java:

this Project is a basic 3D engine built from absolute scratch with Java  
it uses java-core to render 3D shapes and entities to a 2D shape that the screen can display.

## features

| features               | ✅ or ❎ |
| ---------------------- | -------- |
| ZoomIn && Zoom out     | ✅       |
| Rotation               | ✅       |
| Move around the entity | ✅       |
| Camera Movement        | ❎       |
